## How to edit

Open any `.html` file in a text editor and change the page content directly. Shared colours, spacing, typography and responsive styles are in `assets/css/style.css`.

## Deployment

No build step is required. The site can be opened locally by opening `index.html` in a browser.

For Vercel, `vercel.json` is configured to deploy the root folder as a static website and avoid running the old Next.js build script.

## Compliance note

The website is written around non-regulated companionship and practical support. It does not claim registration for regulated activity and does not offer regulated personal care, nursing care or clinical care.
